# ChattingRoom

## Online demo:

[view it on heroku](https://murmuring-brook-22426.herokuapp.com)

## Run it locally:

###For production:
```
npm install
npm run build
node server/server.js
```
And then point your browser to `localhost:3000`


###For debug:
```
npm install
npm run dev
node server/server.js
```
And then point your browser to `localhost:3000`
